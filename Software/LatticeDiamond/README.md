# Lattice Diamond

莱迪思(Lattice)半导体公司提供业界最广范围的现场可编程门阵列（FPGA）、可编程逻辑器件（PLD）及其相关软件，包括现场可编程系统芯片（FPSC）、复杂的可编程逻辑器件（CPLD）,可编程混合信号产品（ispPAC®）和可编程数字互连器件（ispGDX®）。
